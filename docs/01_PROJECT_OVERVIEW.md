# プロジェクト仕様

## 概要

「オリパーマン」はオンラインオリパ（オンラインくじ）サイトの演出システムデモ。
ガチャ抽選の結果を、パチンコ・スロット由来の本格的な演出で盛り上げ、
当落の瞬間まで「脳汁体験」を提供することを目的とする。

このリポジトリは演出ロジックの動作確認用デモであり、課金・在庫・ユーザー管理などの
バックエンドは含まない。抽選結果はクライアント側でランダムに決定される。

## 抽選フロー

1. ホーム画面で「ガチャを引く（100pt）」をタップ
2. レアリティ抽選（`drawResult`） … SSR 0.1% / SR 0.9% / R 9% / N 90%
3. レアリティに応じて発光色を決定（`pickColor`）
4. 発光色に応じて演出ルートを決定（`pickRoute`）
5. オリパーマン本体（メダリオン役物）落下 → 発光示唆
6. 各演出ルートを再生し、当落を判定
7. リザルト画面でレアリティとカードを表示

## 演出ルート

| ルート | 内容 | 当落の決まり方 |
| --- | --- | --- |
| 狙え演出 (`nera`) | スロット風の7揃い。右→中→左の順に停止、テンパイ煽り後にSTOPボタンで左リール停止 | 7揃いで当たり |
| フリーズチャレンジ (`harakiri`) | 3・2・1・0のカウントダウン中にフリーズが発生すれば確定 | フリーズ発生で当たり |
| 次回予告 (`yokoku`) | 液晶＋3リール。小役成立→液晶予告→化け演出 | 予告成立／化けで当たり |
| クルーン演出 (`kurun`) | 3段式の皿。最終皿の当たり穴にボールが入れば当たり | 当たり穴数が多いほど期待度UP |
| 必殺技 (`hissatsu`) | 5秒間の連打でチャージゲージをMAXにする | ゲージMAXで当たり |

## 期待度示唆（発光色）

オリパーマン本体の発光色が期待度を示唆する。

| 色 | 期待度 | レアリティ対応 |
| --- | --- | --- |
| 🔵 青 | 5% | N |
| 🟢 緑 | 20% | R |
| 🔴 赤 | 60% | SR |
| 🌈 虹 | 100%（確定） | SSR |

## デバッグ

画面下部のデバッグパネルから、任意の色・ルート・当落を強制再生できる。
各ボタンは `DBG_BUTTONS` 配列で定義されている。

## 技術スタック

- HTML / CSS / Vanilla JavaScript（フレームワーク不使用）
- [GSAP 3.12](https://gsap.com/)（CDN 読み込み、アニメーション制御）
