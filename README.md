# オリパーマン 演出デモ

オンラインオリパサイト「オリパーマン」の演出システムデモ。
パチンコ・スロット由来の本格演出による圧倒的脳汁体験を再現。

## デモを見る

GitHub Pages で公開中：
https://tatsuhideimao-afk.github.io/oripaman-demo/

## 演出ルート

1. **狙え演出** - スロット風7揃い、テンパイ煽り付き
2. **フリーズチャレンジ** - 3秒以内にフリーズを引け
3. **次回予告** - 液晶＋リール、化け演出付き
4. **クルーン演出** - 3段式皿、当たり穴数で期待度UP
5. **オリパーマン必殺技** - 5秒連打でゲージMAX

## 期待度示唆

オリパーマン本体の発光色で期待度を示唆：
- 🔵 青：5%
- 🟢 緑：20%
- 🔴 赤：60%
- 🌈 虹：100%（確定）
